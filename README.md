# lomansing.github.io

Jekyll site configured for GitHub Pages.

## Local development

1. Install Ruby (with `ridk` / MSYS2 on Windows).
2. Install Bundler:
   `gem install bundler`
3. Install dependencies:
   `bundle install`
4. Run site locally:
   `bundle exec jekyll serve`

Then open `http://127.0.0.1:4000`.
